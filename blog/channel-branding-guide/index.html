Channel Trailer (30-60 seconds)

Script Outline:

  1. **Hook (0-5s):** "You're leaving money on the table if you're not using AI in real estate right now."
  2. **Problem (5-15s):** "Agents spend 30+ hours/week on tasks that AI can do in minutes — follow-ups, lead gen, content creation."
  3. **Solution (15-30s):** "On this channel, I show you exactly which AI tools work, how to use them, and give you the exact prompts."
  4. **Credibility (30-45s):** "I've helped hundreds of agents save 20+ hours/week using these tools."
  5. **CTA (45-60s):** "Subscribe and hit the bell — new videos every Tuesday and Friday."

Trailer Video Specs:

  • Style: Face-to-camera with B-roll cuts of AI tool interfaces
  • Background music: Upbeat tech/corporate (no vocals)
  • CTA overlay at end: "SUBSCRIBE" button animation

Brand Voice & Tone

Voice Attributes

| Attribute | Description |

|-----------|-------------|

| Confident | Magnus knows AI tools — speaks with authority, not arrogance |

| Practical | Every video = actionable takeaways, no fluff |

| Forward-thinking | Always 1-2 years ahead of mainstream adoption |

| Accessible | Explains tech in plain English, assumes audience is tech-curious but not tech-expert |

| Results-oriented | Focus on measurable outcomes: time saved, leads generated, deals closed |

Tone Guidelines

| Do | Don't |

|----|-------|

| Use "you" and "your" frequently | Use passive voice or corporate jargon |

| Give specific tool names + prices | Vaguely say "some AI tool" |

| Show real screen recordings | Use generic stock footage |

| Acknowledge when tools change/break | Over-promise results |

| Be enthusiastic but honest | Sound like a used car salesman |

Phrases to Use

  • "Here's exactly how to do that..."
  • "The tool I use most for this is..."
  • "This one's a game changer..."
  • "In under 5 minutes you can set this up..."
  • "The biggest mistake agents make is..."

Phrases to Avoid

  • "This revolutionary groundbreaking amazing tool" (hyperbole)
  • "Everyone needs to use AI" (condescending)
  • Any reference to specific investment returns without disclaimer

Upload Checklist

For each video, ensure:

  • [ ] Thumbnail: 1280×720px, face visible, <300KB
  • [ ] Title: 60 chars max, primary keyword first, power words
  • [ ] Description: 250+ words, timestamps, links, keywords
  • [ ] Tags: 15-20 tags (mix of broad + long-tail)
  • [ ] Chapter markers: Every major section (auto-add via YouTube Studio)
  • [ ] Cards: 2-3 cards to related videos at 25%, 50%, 75% mark
  • [ ] End screen: Subscribe button + related video (last 20 seconds)
  • [ ] Shorts: Create 1 Short companion per video (top 30 seconds, vertical)
  • [ ] Publish to: Main channel + any connected social accounts
