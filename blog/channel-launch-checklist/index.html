YouTube Channel Launch Checklist

When Google Account is Provided — Immediate Actions

Step 1: Google OAuth with YouTube Scope

# Run outside agent session in terminal
hermes login --provider google
# OR manually authorize at:
# https://accounts.google.com/o/oauth2/v2/auth?scope=youtube.upload+...&prompt=consent

Step 2: Verify Token Has YouTube Scope

The token at ~/.hermes/google_token.json must include: - https://www.googleapis.com/auth/youtube - https://www.googleapis.com/auth/youtube.upload

Step 3: Create Channel (YouTube Studio or API)

  1. Go to YouTube Studio → Create Channel
  2. Name: "AI Tools for Real Estate"
  3. Add channel description
  4. Upload profile picture (1024x1024)
  5. Upload banner (2560x1440)
  6. Add links (website, social)

Step 4: Enable Monetization

  1. YouTube Studio → Monetization
  2. Meet requirements (1K subs + 4K watch hours)
  3. Set AdSense account

Step 5: Upload First Video

  1. Use script #1: "Top 10 AI Tools for Real Estate Agents in 2025"
  2. Create thumbnail via Canva template
  3. Full SEO optimization
  4. Publish
