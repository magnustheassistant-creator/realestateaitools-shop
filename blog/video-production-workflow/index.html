Video Production Workflow: AI Tools for Real Estate

Pre-Production Checklist

Phase 1: Topic Selection & Research (Day 1-2)

  • [ ] Select topic from content calendar
  • [ ] Conduct keyword research
  • [ ] Review competitor videos (watch 3-5 top performers)
  • [ ] Identify unique angle/take on topic
  • [ ] Create outline with main points

Phase 2: Script Writing (Day 2-4)

  • [ ] Write Hook (30 seconds, attention-grabbing)
  • [ ] Write Introduction (1-2 minutes, what viewer learns)
  • [ ] Write Main Content (8-10 minutes, core teaching)
  • [ ] Write Closing + CTA (30-60 seconds)
  • [ ] Write SEO title, description, tags
  • [ ] Review and refine script flow
  • [ ] Time script against target runtime

Phase 3: Visual Asset Preparation (Day 4-5)

  • [ ] Create thumbnail concept (Canva template)
  • [ ] Prepare screen recordings needed
  • [ ] Gather stock footage B-roll options
  • [ ] Prepare any slides or graphics
  • [ ] Create any diagrams or overlays

Production Process

Equipment Needed

Item Recommendation Budget

Tripod Basic phone tripod $25-40
Background Clean wall or virtual backdrop $0-100

Recording Setup

1. Location: Home office or clean professional space
2. Background: Blurred or branded backdrop
3. Lighting: Natural from side + ring light front-facing
4. Camera: Eye level, slightly above
5. Audio: Lavalier mic on, test levels
6. Props: Ready for screen recording reference

Recording Order

  1. Screen recordings first (for tutorials)
  2. On-camera segments second (multiple takes)
  3. B-roll third (if needed additional)
  4. Backup shots (in case of errors)

Recording Tips

  • Write bullet points on sticky notes (not full script)
  • Look at camera lens, not screen
  • Speak naturally, pause between points
  • If error, stop and re-record that segment
  • Keep energy high for hook and closing

Edit Sequence

Step 1: Organize Files (30 min) - Create folder structure: /Project/Assets - Label clips clearly - Export any AI-generated segments - Transcribe audio if needed

Step 2: Rough Cut (1-2 hours) - Import all clips to timeline - Place screen recordings in order - Add on-camera segments - Rough assembly of full video - Check runtime matches target

Step 3: Add Graphics & Text (1 hour) - Add lower thirds - Add callout text - Add chapter markers - Add thumbnail text overlay - Add end screen template

Step 4: Audio Polish (30 min) - Normalize audio levels - Remove background noise - Add music bed (under narration) - Mix voice + music balance

Step 5: Color & Polish (30 min) - Color correct if needed - Add transitions (subtle, minimal) - Check overall flow - Review pacing

Step 6: Export & Upload (30 min) - Export at 1080p minimum, 4K if possible - Format: H.264 for YouTube - Upload to YouTube Studio - Add thumbnail - Write SEO description and tags - Schedule or publish

Quality Standards Checklist

Pre-Review (Before Publishing)

  • [ ] Hook is compelling (watch first 30 seconds 5x)
  • [ ] Audio is clear and consistent
  • [ ] Visuals are stable (no shaky cam without purpose)
  • [ ] Text is readable at all screen sizes
  • [ ] No dead air or awkward pauses
  • [ ] CTA is clear and actionable
  • [ ] Description is SEO-optimized
  • [ ] Tags are relevant and specific
  • [ ] Thumbnail matches content and is eye-catching
  • [ ] End screen is properly placed
  • [ ] Chapters/timestamps are accurate

Technical Check

  • [ ] Resolution: 1080p or higher
  • [ ] Bitrate: 8-12 Mbps for 1080p
  • [ ] Audio: 128kbps minimum
  • [ ] File format: MP4
  • [ ] Aspect ratio: 16:9

Production Timeline (Per Video)

Day Task Time

Day 2 Script writing 2-3 hours
Day 3 Visual prep + thumbnail 1-2 hours
Day 4 Recording (screen + camera) 1-2 hours
Day 5 Video editing 3-4 hours
Day 6 Review + revisions + upload 1-2 hours

Total time per video: 10-17 hours

Batch Production Strategy (Recommended)

For efficiency, batch similar tasks: - Batch 1: Research all 10 videos in one session (1 day) - Batch 2: Write all 10 scripts in one session (2-3 days) - Batch 3: Record all screen segments (1-2 days) - Batch 4: Record all on-camera segments (1-2 days) - Batch 5: Edit all 10 videos (1-2 weeks) - Batch 6: Create all thumbnails (1-2 days) - Batch 7: Upload and schedule all (1 day)

Total batch time: 3-4 weeks for 10 videos
