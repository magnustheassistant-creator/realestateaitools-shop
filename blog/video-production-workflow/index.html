Pre-Production (1-2 days before filming)

Day 1: Script & Preparation

  1. **Select video topic** from content calendar
  2. **Review/adapt script** — adjust examples to current market, add any new AI tool updates
  3. **Gather assets:**
  4. AI tool screenshots (capture current interfaces)
  5. Stock footage links (if using)
  6. Any market data/stats to reference
  7. **Create thumbnail** (see thumbnail-concepts.md)
  8. **Prepare background** — clean desk, good lighting, professional but friendly setting
  9. **Test audio** — use external mic if available, test for echo/reverb

Day 2: Filming Checklist

  • [ ] Recording software open (OBS Studio / Camo / Loom)
  • [ ] Scripts open in teleprompter (Teleprompter.app / PowerPoint)
  • [ ] All tool interfaces loaded and ready to screen-share
  • [ ] Water nearby, lights on, phone on silent
  • [ ] Recording: Intro, then body, then close (allows worst-to-best energy curve)

Recording Order

  1. **Record intro** (first 30 seconds) — multiple takes, best energy
  2. **Record main content** — sections in order, can do retakes per section
  3. **Record outro/CTA** — second-to-last, warm closing energy
  4. **Record placeholder for B-roll** — mark where screen recordings will go

Best Practices

  • **Eye line:** Look at camera, not at screen below
  • **Energy:** Talk like you're explaining to ONE person, not a crowd
  • **Pace:** Slightly slower than feels natural — editing will speed it up
  • **Retakes:** It's OK to pause and restart a sentence — you can cut later
  • **B-Roll:** Note timestamps where you'll insert screen recordings

Post-Production

Editing Pipeline (Per Video)

Phase 1: Initial Assembly (30-45 min)

  1. Import all footage into editor (DaVinci Resolve free / CapCut / Adobe Premiere)
  2. Assemble in order: Intro → Body sections → Outro
  3. Remove obvious bad takes, long pauses, ums/uhs
  4. Insert B-roll placeholders where needed

Phase 2: B-Roll & Graphics (30-45 min)

  1. Record screen recordings of AI tools (Loom / Camtasia / built-in recorder)
  2. Insert screen recordings at marked timestamps
  3. Add text overlays: key points, tool names, timestamps
  4. Add lower thirds / name badge (Magnus - AI Tools for Real Estate)

Phase 3: Polish (15-30 min)

  1. Add intro animation (Canva animated intro or pre-made template)
  2. Add outro/end screen with subscribe button (YouTube Studio can auto-add)
  3. Color correct footage for consistency
  4. Add captions (YouTube auto-captions + manual review)
  5. Add background music (Epidemic Sound / Artlist / YouTube Audio Library)

Phase 4: Export & Upload (15 min)

  1. Export at 1080p or 4K, H.264 codec, MP4 format
  2. Upload to YouTube Studio
  3. Add title (use keyword + year + format)
  4. Add description (use template)
  5. Add tags (primary keyword + secondary + related)
  6. Add timestamp chapters
  7. Add custom thumbnail (upload from Canva)
  8. Set visibility: Public / Scheduled

AI Voice Workflow

  1. Write/provide script (already done in video-scripts/)
  2. Generate AI voice-over using selected tool
  3. Record screen capture of AI tool being used
  4. Sync audio with screen recordings in editor
  5. Add intro/outro webcam footage (optional for faceless channel)
  6. Export and upload

Tools & Software Stack

Recording

  • **OBS Studio** (free) — Screen + webcam recording
  • **Camtasia** (paid) — Screen recording + basic editing
  • **Loom** (free tier) — Quick screen recordings
  • **Zoom** (free) — Record separate screenshare calls

Editing

  • **DaVinci Resolve** (free) — Professional editing, color grading
  • **CapCut** (free) — Easy editing, good for beginners
  • **Adobe Premiere** (paid subscription) — Industry standard
  • **Canva** (free tier) — Video editing, thumbnails, animated intros

Screen Recording

  • **Loom** (free) — Browser + app recording
  • **Camtasia** (paid) — Advanced screen recording + editing
  • **OBS Studio** (free) — Full screen/window capture
  • **ShareX** (free, Windows) — Lightweight screen capture

Asset Creation

  • **Canva** (free tier) — Thumbnails, intros, lower thirds, graphics
  • **Figma** (free) — Custom graphics and overlays
  • **Pexels / Pixabay** (free) — Stock footage for B-roll
  • **Epidemic Sound / Artlist** (paid) — Background music

Quality Checklist (Before Publishing)

Video

  • [ ] Watch full video at 1x speed, no skips
  • [ ] Check all B-roll is properly synced
  • [ ] Verify all tool names and versions are accurate
  • [ ] Confirm no dead links in description
  • [ ] Check audio levels are consistent
  • [ ] Verify captions are accurate

Description

  • [ ] Title has primary keyword and year
  • [ ] Description has 150+ words with keywords
  • [ ] Timestamps are accurate and match video
  • [ ] Tags include primary + secondary + related keywords
  • [ ] Hashtags formatted correctly (#keyword)
  • [ ] Free resource link works (if applicable)
  • [ ] CTA is clear

Thumbnail

  • [ ] Custom thumbnail uploaded (not auto-generated)
  • [ ] Text is readable at small sizes
  • [ ] Consistent brand colors used
  • [ ] Magnus brand is visible

SEO

  • [ ] Title includes searchable keyword
  • [ ] Description includes primary keyword in first sentence
  • [ ] Tags cover main topic and related keywords
  • [ ] Video is added to relevant playlist
  • [ ] End screen cards link to 1-2 related videos

Scalability Options

  1. **Hire video editor** (Fiverr / Upwork / onlinejobs.ph) — $50-200/video
  2. **Outsource script writing** — Provide outlines, contractor expands
  3. **Hire virtual assistant** — Create thumbnails, upload videos, manage comments
  4. **AI avatar option** — Synthesia/DId for completely faceless videos
  5. **Repurpose content** — 1 YouTube video → blog post → LinkedIn → email newsletter → TikTok/Reels shorts
